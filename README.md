# Sector 317

Sector 317 booking and management system built with React + Vite.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deploy

Connected to Vercel via GitHub. Push to `main` to deploy automatically.
